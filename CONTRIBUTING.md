# Contributing to HunterTestPrep

Thank you for your interest in contributing to HunterTestPrep! This project aims to help students prepare for Hunter College Campus Schools admission tests.

## How to Contribute

### Reporting Issues

If you find errors in questions, answers, or documentation:

1. Check existing issues to avoid duplicates
2. Create a new issue with a clear title
3. Include the file path and specific error
4. Suggest the correct information if known

### Adding New Questions

When contributing new practice questions:

1. Follow the existing format in the `questions/` directory
2. Ensure questions are original (not copied from copyrighted materials)
3. Provide answer keys with explanations in the `solutions/` directory
4. Test your questions for accuracy

### Question Guidelines

- Questions should be grade-appropriate (5th-6th grade level)
- Include a variety of difficulty levels
- Cover all topics listed in the test format documentation
- Do not use questions from actual Hunter exams

### Code Contributions

For contributions to the web application or scripts:

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Test thoroughly
5. Submit a pull request with a clear description

### Documentation

Help improve documentation by:

- Fixing typos and grammatical errors
- Clarifying instructions
- Adding helpful study tips
- Updating test format information

## Code of Conduct

- Be respectful and constructive
- Focus on educational value
- Do not share actual exam questions
- Credit original sources when applicable

## Questions?

Open an issue if you have questions about contributing.
